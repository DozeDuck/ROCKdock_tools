Quazip library
==============

To replace the Quazip library copy your version in the following directories:

Windows:
- <INSTALL_DIR>

Linux:
- <INSTALL_DIR>/lib

macOS:
- <APP_DIR>/Contents/Frameworks

Quazip source code
==================
The Quazip source code can be obtained from:
https://sourceforge.net/projects/quazip/

or from our server:
http://www.biosolveit.de/ext_src/quazip-src.zip

In the unexpected case of unavailability of the server please send an email at
ext_src@biosolveit.de. We will ensure access to the requested sources as soon as
possible.
